#  Цифровая физика: модель соскальзывания цепочки
Some Arduino Projects for Digital Physics
Проекты для Arduino по работе Цифровая физика Степана Воротницкого
Цель работы применить разработанный автором микроконтроллерный комплекс на платформе Arduino для математического моделирования и экспериментального исследования физических процессов, модели которых обычно строятся с помощью методов высшей математики, недоступных школьнику. Традиционно для компьютерного моделирования таких процессов с помощью методов высшей математики записываются уравнения, которые решаются аналитически или численно.Gредлагается вместо этого переходить к дискретным величинам до записи этих уравнений, рассматривая сами физические процессы в дискретном мире. Мы будем сравнивать результаты моделирования с экспериментом, который также будет поставлен с помощью компьютерной системы. 

Программа осуществляет моделирование соскальзывания цепочки со стола. Описание задачи - смотри в прикрепленнном файле Description1.pdf
